Instruções iniciais - Organização do projeto

SCRUM MASTER:

  • Criar repositório no github
  • Adicionar os integrantes do grupo como colaborador do repositório (Basta entrar na pagina do repositório > Settings >  Collaborators )
  • Criar board no Trello
  • Adicionar os integrantes no board e dar permissão para editar
  • Criar 5 colunas: 

Backlog Produto (todos os requisitos do projeto, separados em tarefas/cards)

Backlog Sprint (tarefas da sprint atual)

TO DO (tarefas que já podem ser executadas)

DOING (tarefas que estão sendo executadas)

DONE (tarefas finalizadas)

  • Adicionar todos os requisitos do projeto, 1 card para cada tarefa.


DEV TEAM:

  • Criar conta no github e aceitar convite para ser colaborador do repositório do projeto
  • Criar conta no trello, aceitar convite para participar do board.
